Crop Yield Predictor

A Machine Learning pipeline predicting crop yield (hg/ha) from environmental inputs (Rainfall, Temperature) and agricultural inputs (Pesticides) using a Gradient Boosting Regressor.

The Problem

Crop yield is highly volatile due to shifting weather and inconsistent resource use, making it hard for farmers and policymakers to plan for food security.

Engineering & Data Science Solutions

  • Data Integrity: Built a cleaning pipeline to isolate and remove non-numeric garbage values in the raw Rainfall dataset.
  • High-Cardinality Features: Applied One-Hot Encoding on complex categorical variables to preserve geographical context without data loss.
  • Model Optimization: Benchmarked 5 regression algorithms. Gradient Boosting achieved 96% accuracy (R²), significantly outperforming the baseline.

Business Simulation & Analysis

  • Anomaly Detection: Identified 155 anomalies from 5,649 test samples using Z-score analysis.
  • Intervention Scenario: Modeled a real-world simulation targeting low-yield farms (<30,000 hg/ha).

Tech Stack

Python, Pandas, NumPy, Scikit-Learn, Seaborn, Matplotlib
