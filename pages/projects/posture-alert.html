Posture Alert App

A webcam-based web application that detects spinal misalignment in real-time using geometric analysis — no additional hardware required.

The Challenge

Slouching is prevalent among computer users, but hardware-based solutions are expensive and intrusive. Standard software models also face tracking issues when features are occluded.

Engineering Solutions

  • Occlusion Handling: Standard pose models lose tracking when users wear headphones (ears get occluded). Tracking was re-implemented on Eye Landmarks instead of ears, maintaining detection continuity.
  • Noise Filtering: A Calibration Lock Algorithm was built to calculate a normalized shoulder-width baseline for the primary user. It ignores any detected figure smaller than 60% of the calibrated size, eliminating false alerts from background movement.
  • Depth Invariance: The system relies on relative geometric ratios rather than raw pixel distances, allowing it to function accurately regardless of camera distance.

Features

  • Live slouch alerts
  • Event logging
  • Exportable session reports

Tech Stack

Python, MediaPipe (Pose Estimation), OpenCV2, Streamlit, Pandas
