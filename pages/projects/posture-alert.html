Posture Alert App

A webcam-based web application that detects spinal misalignment in real-time using geometric analysis, requiring no additional hardware.

The Challenge

Slouching is widespread among computer users, but hardware solutions are often intrusive or expensive. A major technical hurdle was occlusion; standard pose models stop tracking when users wear headphones. I solved this by re-implementing landmark tracking on eye landmarks instead of ears.

The Solution

I developed a system using MediaPipe for pose estimation and OpenCV for geometric analysis. To eliminate false alerts from background movement, I implemented a Calibration Lock Algorithm that ignores any figures smaller than 60% of the calibrated user's shoulder-width baseline.

Tech Stack

Python, MediaPipe, OpenCV2, Streamlit, Pandas
