Public Transport Route Planner

A web-based Route Planner & Tracker that aggregates static schedules and live GPS vehicle data to provide optimized routes, real-time ETAs, and statistical on-time reliability metrics.

The Challenge

Urban commuters face uncertainty due to unreliable public transport schedules, unpredictable delays, and no centralized real-time disruption feed.

Engineering Solutions

  • Hybrid Data Sync: Engineered a fail-safe ETA engine that prioritizes live Firebase GPS data, with automatic fallback to static MySQL schedules if live data is unavailable — ensuring zero downtime on ETA estimates.
  • Geospatial Visualization: Dynamic Leaflet polylines render accurate route corridors by fetching ordered stop coordinates from the database.
  • Reliability Quantification: Created "On-Time Insight," a custom statistical metric that compares historical vs. scheduled arrival times to calculate a reliability percentage per route.

Features

  • Account registration and login system
  • Route search between stops
  • Live vehicle arrivals panel with GPS indicator
  • Service disruption notices panel
  • Interactive map with real-time route rendering

Tech Stack

PHP (Laravel Framework), Eloquent ORM, MySQL, Blade Templates, Vite, Tailwind CSS, Leaflet.js, Firebase
