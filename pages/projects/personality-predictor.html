Personality Predictor App

A custom-trained Computer Vision Neural Network that analyzes clothing attributes from images and maps them to OCEAN trait scores.

The Concept

Predicting users' personality type based on their visual presentation and clothing choices, utilizing the OCEAN (Big Five) personality model.

The Solution

A deep learning architecture designed to extract visual features from user-uploaded images. These features are then passed through a specialized classification network that correlates fashion choices with the five major personality traits: Openness, Conscientiousness, Extraversion, Agreeableness, and Neuroticism.

Tech Stack

Python, HTML, CSS, JavaScript, PyTorch, Flask
